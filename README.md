Este repositório contém o código de um site simples que reúne links para os portfólios dos alunos do grupo.

O objetivo do site é facilitar o acesso aos portfólios individuais, centralizando todos os links em um único lugar para visualização e compartilhamento.

Cada link direciona para o portfólio pessoal de um aluno, onde estão disponíveis seus projetos, informações e contatos.
